#第三次project之二

##题目
- 代码实现Dijkstra #4（基于循环桶）

##思路
- 最小值：在第一个非空的list内的set随便找一个并且删除
- 变化值： 变distance并且set内删除增加
- 指向桶的指针循环，存入桶的元素也循环
- 读入图的时候搞一个最大value出来
- 函数输入变量包括图和最大边（主要是比较懒不高兴再算最大边）

##存储
- 桶：桶列表拿list存储，单个桶使用set存储
- 图：字典套字典
- 父节点（parent）：字典
- 距离：字典
- 路径：字典套list
- 要考虑点无法到达的情况，如果不考虑，很容易因为V-X这样的判决进入死循环，所以如果所有桶空那么也提前结束循环
