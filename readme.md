# 通信网理论基础第一次Project
- 使用环境为Python3.6.4

## 题目要求
- 设计并实现求两点间所有路径的算法。

## 一般求解算法过程
- 建模――设计――证明――分析――实现（伪代码与数据结构）――编程（python）。

## 思路
- 初始思路：使用DFS，但是遍历到目的地点就停止，并且将路径入栈，停止后出栈，即从倒数第二个点继续寻路，直到倒数第二个点除它的上一层外其余均已考虑。
- 需要另外加一个栈来存储路径，由于有 if u in stack 这种语句，所以不需要是否入栈的标识符。
- “已考虑”这种思路需要另外的标识符，可以用一个dict实现，即已考虑的在dict对应点的list内。

## 输入输出
- 输入一张图以及起始点、结束点，输出多条字符串，指示路径。
